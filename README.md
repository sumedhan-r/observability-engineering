# Observability Engineering

Personal notes and practical examples for implementing observability in software projects.

## Repository Structure

- **`knowledge/`** - Notes and concepts from 'Observability Engineering'
  - `01_observability_engineering_foundation.md` - Foundational concepts
  - `02_observability_engineering_telemetry.md` - Telemetry principles

- **`telemetry/`** - Python example project demonstrating telemetry implementation
  - Shows best practices for traces, metrics, and logs
  - Ready-to-use patterns for observability setup

## Getting Started

1. Read the knowledge notes to understand observability concepts
2. Explore the `telemetry/` project for practical implementation examples
